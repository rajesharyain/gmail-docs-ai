# Changelog

All notable changes to Gmail Docs AI will be documented in this file.

## 0.1.0 - Unreleased

### Added

- Created the standalone Gmail Docs AI project from the existing macOS
  menu-bar foundation.

- Added Gmail-focused documentation, security notes, and roadmap.
